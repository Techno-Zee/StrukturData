# Stack 🗄️

adalah konsep memasukkan data dimana data yang terakhir masuk akan diambil terlebih dahulu. seperti halnya sebuah wadah biskuit dimana biskuit terakhir yang dimasukkanlah yang paling mungkin diambil terlebih dahulu

## Push ⏬
operasi meletakkan elemen/item kedalam sebuah stack (wadahnya).

## If Full
untuk mengecek apakah wadahnya penuh/belum

## Pop
mengeluarkan elemen/item keluar dari stack
