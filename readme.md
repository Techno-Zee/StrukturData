# Struktur Data
mata kuliah yang  menjelaskan bagaimana cara mengorganisir, menyimpan, dan mengelola data secara efisien dalam memori komputer.

namun dalam semester 2 ini kita terlebih dahulu mengulas kembali beberapa materi yang sebelumnya ada di algoritma pemrograman, sembari mempelajarinya lebih dalam.

## Pengelolaan Folder

```

 ```
